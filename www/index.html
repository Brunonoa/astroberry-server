Welcome to Astroberry Server

Astroberry Server is a ready to use astronomy desktop system for Raspberry Pi 3 Model B and B+
It is a standalone system for controlling astronomy equipment supported by INDI server.

Connect to your Astroberry Server

If you want to use encrypted connection install and trust security certificate.
For extended security you may consider installing your own certificates.
